# superset_dashboard_embedding (Resource)

Manages embedded dashboard configuration in Apache Superset. When a dashboard is embedded, Superset generates a UUID that can be used to construct an embed URL for iframe integration.

## Example Usage

```terraform
resource "superset_dashboard_embedding" "example" {
  dashboard_id    = superset_dashboard_import.example.dashboard_id
  allowed_domains = ["https://app.example.com", "https://portal.example.com"]
}

output "embedded_dashboard_uuid" {
  value = superset_dashboard_embedding.example.uuid
}
```

### Allow all domains

```terraform
resource "superset_dashboard_embedding" "public" {
  dashboard_id    = 42
  allowed_domains = []
}
```

## Import

Import using the dashboard numeric ID:

```shell
terraform import superset_dashboard_embedding.example 42
```
